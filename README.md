# OnlyStudents Agent

> 基于 LangChain + 阿里云百炼的 AI 智能助手，为 only-students 平台提供对话、搜索和内容理解能力。

## 项目概述

OnlyStudents Agent 是一个独立部署的 Python 微服务，采用 RAG（检索增强生成）架构，使 AI 能够：
- 理解站内笔记内容
- 基于笔记回答用户问题
- 支持 `@AI助手` 评论触发
- 实现类 Grok 的对话体验

## RAG 流程

### 完整 RAG 流程图

```
┌─────────────────────────────────────────────────────────────────────────────┐
│                              RAG 检索增强生成                               │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  用户问题: "有什么Python入门笔记推荐？"                                      │
│                          │                                                 │
│                          ▼                                                 │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  1. Query Embedding (通义千问)                                        │   │
│  │     - 文本 → 1536维向量                                              │   │
│  │     - 模型: tongyi-embedding-vision-plus                             │   │
│  └────────────────────────────┬────────────────────────────────────────┘   │
│                                │                                            │
│                                ▼                                            │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  2. 向量检索 - Milvus (召回 Recall)                                  │   │
│  │     - Top-50 相关笔记                                                │   │
│  │     - 索引类型: HNSW / IVF                                           │   │
│  │     - 距离度量: Cosine Similarity                                    │   │
│  └────────────────────────────┬────────────────────────────────────────┘   │
│                                │                                            │
│                                ▼                                            │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │  3. Prompt 构建 + LLM 生成 (Generation)                              │   │
│  │     - 系统提示词 + 上下文 + 用户问题                                   │   │
│  │     - 模型: 通义千问 qwen3.5-flash                                   │   │
│  │     - 流式输出 (SSE)                                                  │   │
│  └────────────────────────────┬────────────────────────────────────────┘   │
│                                │                                            │
│                                ▼                                            │
│                      ┌─────────────────┐                                    │
│                      │   返回回答      │                                    │
│                      │  基于笔记内容    │                                    │
│                      └─────────────────┘                                    │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

## 核心模块说明

### 1. RAG 核心 (services/rag.py)

负责检索增强生成的核心逻辑：
- 向量检索
- 上下文构建
- Prompt 组装
- LLM 调用

### 2. 会话记忆 (services/memory.py)

基于 Redis 的对话记忆管理：
- 多轮对话上下文
- Session 管理
- 自动过期清理

### 3. 笔记工具 (tools/note_search.py, note_reader.py)

LangChain Tools 实现：
- `search_notes`: 搜索站内笔记
- `read_note`: 读取笔记详情

### 4. 向量处理 (vector/)

- `embedder.py`: 通义千问 Embedding 封装
- `client.py`: Milvus 客户端封装
- `ingest.py`: 笔记数据导入

## 快速开始

### 1. 启动依赖服务

```bash
# 启动 Milvus + Ollama + Redis
docker compose up -d

# 检查服务状态
docker compose ps
```

### 2. 安装依赖

```bash
cd only-students-agent
pip install -r requirements.txt
```

### 3. 同步笔记数据

```bash
python scripts/sync_notes.py
```

### 4. 启动服务

```bash
uvicorn app.main:app --reload --port 8000
```
